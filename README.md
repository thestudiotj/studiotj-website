# StudioTJ Portfolio Website

Next.js 14 portfolio site for StudioTJ. Includes:
- Portfolio gallery fed by the photo sorter manifest
- Blog/notes via MDX files
- Shop via Shopify Storefront API (reuses your existing Shopify)
- Affiliate pages (coming next)

## Setup

```bash
# 1. Install dependencies
npm install

# 2. Configure environment
cp .env.example .env.local
# Edit .env.local with your Shopify credentials

# 3. Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Adding your photos

1. Run the photo sorter: `python main.py --folder /your/photos --output manifest.json`
2. Copy `manifest.json` to `studiotj-website/public/manifest.json`
3. Copy your photos to `studiotj-website/public/photos/`
4. The portfolio page will auto-populate from the manifest

## Writing blog posts

Create `.mdx` files in `content/blog/` with this frontmatter:

```mdx
---
title: "Your post title"
date: "2026-02-20"
excerpt: "One line preview shown in the list"
tags: ["process", "netherlands"]
---

Your content here...
```

## Connecting Shopify

Get your Storefront API token from:
Shopify Admin → Settings → Apps → Develop apps → Storefront API

Add to `.env.local`:
```
SHOPIFY_STORE_DOMAIN=your-store.myshopify.com
SHOPIFY_STOREFRONT_TOKEN=your-token
```

## Deploying to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Set env vars in Vercel dashboard or via CLI
vercel env add SHOPIFY_STORE_DOMAIN
vercel env add SHOPIFY_STOREFRONT_TOKEN
```

## Project structure

```
src/
  app/
    page.tsx           ← Homepage
    portfolio/         ← Gallery + collection filtering
    blog/              ← Blog listing + post pages
    shop/              ← Shopify-powered shop
  components/
    Nav.tsx            ← Fixed nav with scroll behaviour
    Footer.tsx
    PortfolioGallery.tsx ← Masonry grid + lightbox
  lib/
    manifest.ts        ← Reads photo sorter manifest.json
    blog.ts            ← Reads MDX blog posts
    shopify.ts         ← Shopify Storefront API
content/
  blog/                ← MDX blog posts go here
public/
  manifest.json        ← Generated by photo sorter
  photos/              ← Your actual photos go here
```
